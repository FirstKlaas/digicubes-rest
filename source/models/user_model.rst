id - The primary key field
^^^^^^^^^^^^^^^^^^^^^^^^^^

.. code-block:: JSON

    {
        "pk" : true,
        "null" : false,
        "unique" :true,
        "class" : "Intfield"
    }

The ``id`` attribute is the primary key of the model.
Its value is autogenerated. After creating a new user
instance, the id can be read.

.. code-block:: python

    new_user = User.create(login="ratchet")
    print(new_user.id)
